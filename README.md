# Machine Learning Based Disaster Prediction and Early Warning System

A Python mini-project using Random Forest and Streamlit to predict disaster risk based on weather and soil data.

## Run Steps
1. Install dependencies  
   `pip install -r requirements.txt`

2. Train the model  
   `python train_model.py`

3. Launch the Streamlit app  
   `streamlit run app.py`

Model accuracy: ~96%
